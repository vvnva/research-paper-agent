# AI Research Paper Finder

AI Research Paper Finder - это агентная система, которая помогает студентам и исследователям находить релевантные научные статьи по теме исследования.

Система автоматически ищет статьи в научных базах, оценивает их релевантность и предоставляет краткие summary, чтобы ускорить первичный этап literature review.

PoC реализован в виде Telegram-бота.

---

## Проблема

Когда человек начинает исследовать новую тему, он сталкивается с рядом проблем:

- поиск статей занимает много времени
- результаты поиска часто слишком широкие или нерелевантные
- необходимо вручную читать множество abstract'ов
- сложно быстро понять, какие статьи наиболее важны для старта исследования

В результате исследователь тратит часы на первичный поиск литературы.

AI Research Paper Finder автоматизирует этот процесс.

---

## Что делает PoC

На демо система:

1. Принимает тему исследования от пользователя
2. Переформулирует запрос для более эффективного поиска
3. Ищет научные статьи через внешний API
4. Анализирует метаданные и abstract статей
5. Ранжирует статьи по релевантности
6. Показывает пользователю топ наиболее релевантных работ
7. Генерирует краткое описание каждой статьи

Пользователь получает компактную подборку статей для старта исследования.

---

## Out of scope

Данный PoC не предназначен для полного автоматического literature review.

Система:

- не анализирует полный текст PDF статей
- не гарантирует научную новизну найденных работ
- не выполняет глубокий академический анализ
- не заменяет научного руководителя или эксперта
- не анализирует сотни статей одновременно
- не делает автоматические цитирования

PoC предназначен для ускорения начального этапа поиска литературы.

---

## Архитектура (кратко)

Система состоит из нескольких модулей:

- Query Agent — улучшает пользовательский запрос
- Retrieval Tool — ищет статьи через arXiv API
- Ranking Agent — оценивает релевантность результатов
- Summarization Agent — создает краткие summary
- Session Memory — хранит контекст пользователя

PoC реализован в виде Telegram-бота

- перейти по ссылке на оригинальную публикацию
